#Predicción de Riesgo Crediticio

## Descripción
Este proyecto tiene como objetivo predecir el riesgo de incumplimiento de pago de clientes, utilizando técnicas de minería de datos y aprendizaje supervisado.

Se desarrolla como una aplicación web básica con Flask que presenta la formulación del problema y la estructura inicial del proyecto.

##  Objetivo General
Desarrollar un modelo que permita identificar clientes con alto riesgo de incumplimiento de pago, apoyando la toma de decisiones en entidades financieras.

 ## Objetivos Específicos
- Analizar el comportamiento financiero de los clientes.
- Identificar variables relevantes en el riesgo crediticio.
- Preparar los datos (limpieza e imputación).
- Aplicar un modelo de clasificación.
- Evaluar el modelo con métricas como Accuracy, Recall y F1-score.

##  Tecnologías Utilizadas
- Python
- Flask
- HTML / CSS
- Git & GitHub
